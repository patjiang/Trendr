About TRENDR

TRENDR allows you to visualize and analyze trending Reddit keywords across the top 50 subreddits from 2011-2024. It is a powerful tool for market research and uncovering organic customer insights.

Features

  • Visualize keywords as time series graphs.
  • Analyze keyword trends across top subreddits.
  • Monitor upvote/downvote changes for specific keywords.
  • Filter visualizations by subreddit categories.

Database Architecture

TRENDR's backend database is powered by PostgreSQL on AWS, with Python and Django for data interaction. The schema includes tables for Subreddits, Posts, Keywords, and Post-Keyword relationships.

Example SQL Queries

-- Get keywords for posts with over 100,000 upvotes:
SELECT pk_word 
FROM post_keyword 
JOIN post 
ON post_keyword.pk_post_id = post.p_post_id 
WHERE post.score > 100000;

-- Get URL of the most popular post for a specific keyword:
SELECT url 
FROM post 
JOIN post_keyword
ON post.p_post_id = post_keyword.pk_post_id 
WHERE post_keyword.pk_word = 'lol'
ORDER BY post.score DESC 
LIMIT 1;
