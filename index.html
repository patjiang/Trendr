TRENDR

Unlock insights from Reddit data like never before.

With TRENDRv1, the user will be able to perform the following visualizations/analyses:

  • Visualize keywords from posts as a time series on a line graph by raw instance count.
  • Visualize keywords on a table by instance count percentile.
  • For a specific keyword, see how the upvote and downvote counts change for posts containing that keyword over time.
  • Optionally limit the visualizations by any subset of the top 50 subreddits.

Our database will consist of four main entities organized from the original data obtained from the Reddit API:

  • Subreddits: Contains information about each subreddit.
  • Posts: Contains information about individual posts.
  • Post Type: Information about the categorization of a post.
  • Keywords: Table containing information about keywords in posts.

  • SELECT Queries for extracting specific information, such as top subreddits or most frequent keywords.
